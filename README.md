# Netflix Data Analysis

## Project Overview
Analyzed Netflix movies and TV shows dataset using Python to find trends, patterns, and insights from Netflix content.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Dataset
Netflix Titles dataset containing information about movies and TV shows, including:
- Title
- Type
- Country
- Release Year
- Rating
- Genre
- Duration

## Work Done
- Data Cleaning
- Missing Value Handling
- Data Formatting
- Exploratory Data Analysis (EDA)
- Data Visualization

## Analysis Performed
- Movies vs TV Shows comparison
- Top content producing countries
- Popular genres analysis
- Ratings distribution
- Year-wise content growth

## Key Insights
- Netflix contains more movies than TV shows
- USA produces the highest number of Netflix content
- TV-MA is one of the most common ratings
- Netflix content growth increased significantly over the years

## Visualizations
Created graphs using Matplotlib to understand:
- Content distribution
- Country trends
- Genre popularity
- Growth trends

## Conclusion
This project helped in understanding real-world data analysis workflow using Python, including cleaning data, analyzing patterns, and presenting insights visually.
